Introduction sur la partie licence et rappel de l'articile de présentation

Lorsque que Microsoft Intune est utilisé pour la gestion des terminaux de l'entreprise il est nécessaire d'intégrer les appareils gérés par ce dernier dans Microsoft Defender for Endpoint. Même si ces deux produits Microsoft fonctionnent ensembles nativement, il est nécessaire d'effectuer plusieurs actions de configurations afin de profiter de l'ensemble des fonctionnalités proposées par MDE.

Une fois les étapes décrites ci-après effectuées, l'intégration des périphériques dans Microsoft Defender for Endpoint et donc dans le "Security Center" se fera de manière totalement transparente.
