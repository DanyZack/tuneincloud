Étape 1 : Accédez à la console Endpoint Manager Connectez-vous à la console Endpoint Manager avec un compte d'administrateur et accédez à la section "Windows Autopilot deployment". Dans cette section, cliquez sur "Deployment Profiles".

Étape 2 : Sélectionnez un profil d'enrôlement Windows Autopilot Sélectionnez le profil d'enrôlement Windows Autopilot pour lequel vous souhaitez configurer l'ESP. Cliquez sur "Properties" pour accéder aux options de configuration.

Étape 3 : Accédez à la page de configuration de l'ESP Dans la section "Out-of-box experience (OOBE)" du profil d'enrôlement, cliquez sur "Enrollment Status Page (ESP) settings" pour accéder à la page de configuration de l'ESP.

Étape 4 : Configurez les options de l'ESP Dans la page de configuration de l'ESP, vous pouvez configurer plusieurs options pour la gestion des appareils Windows Autopilot.

- Afficher ou masquer la page de configuration de l'ESP : vous pouvez choisir d'afficher ou de masquer la page de configuration de l'ESP pendant le processus d'enrôlement. Si vous choisissez de masquer la page de configuration de l'ESP, les utilisateurs ne verront pas l'ESP et ne pourront pas voir les informations de statut de l'enrôlement.

- Afficher ou masquer les informations de l'appareil : vous pouvez choisir d'afficher ou de masquer les informations de l'appareil sur la page de configuration de l'ESP. Si vous choisissez de masquer les informations de l'appareil, les utilisateurs ne pourront pas voir les détails de l'appareil, tels que le nom d'utilisateur, l'adresse IP, le numéro de série, etc.

- Configurer le message de bienvenue : vous pouvez personnaliser le message de bienvenue sur la page de configuration de l'ESP pour fournir des informations supplémentaires aux utilisateurs, telles que des instructions pour l'enregistrement de l'appareil ou des informations de support.

- Configurer les options de redémarrage : vous pouvez choisir de redémarrer automatiquement l'appareil après l'enrôlement et/ou de forcer le redémarrage en cas de problème lors de l'enrôlement.

- Configurer le comportement de l'ESP en cas d'échec de l'enrôlement : vous pouvez spécifier si l'ESP doit continuer à s'afficher ou si l'appareil doit être redémarré en cas d'échec de l'enrôlement.

- Configurer la durée d'affichage de l'ESP : vous pouvez spécifier pendant combien de temps l'ESP doit rester visible après l'enrôlement.

Étape 5 : Enregistrez les modifications Une fois que vous avez terminé de configurer les options de l'ESP, cliquez sur "Save" pour enregistrer les modifications.

Et voilà, vous avez configuré l'ESP pour votre profil d'enrôlement Windows Autopilot en utilisant toutes les possibilités de gestion disponibles.
